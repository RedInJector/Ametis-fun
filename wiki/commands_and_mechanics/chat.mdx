# Чат

За замовчуванням основним каналом в чаті стоїть локальний, тобто якщо ви просто пишете у чат це локальний чат, але ви можете це змінювати пишучи одну з команд в низу, проте без повідомлення, тим самим змінювати канал за замовчуванням особисто для себе.

**/л** (повідомлення) - написати повідомлення в локальний чат (радіус 100 блоків)

**/г** (повідомлення) - написати повідомлення в глобальний чат (працює між вимірями)

**/пп** (гравець) (повідомлення) - написати приватне повідомлення до гравця (працює між вимірами) <br/>
 */vmessage, /msg, /tell, /whisper, /pm, /w, /t - також інші робочі варіанти*

**/в** (повідомлення) - відповісти на останнє повідомлення в приватних повідомленнях <br/>
*/r, /reply - також інші робочі варіанти*

**/ігнор** (гравець) - ігнорувати певного гравця, тим самим він не зможе відправляти вам повідомлення <br/>
*/vignore, /і, /ignore - також інші робочі варіанти*

Решту можливих налаштувань ви можете виконати через меню /chatsettings, або в лобі вибравши повторювач та натиснувши праву кнопку миші.

## Перехід між серверами

**/lobby** - повернутись в лобі севрера

## Спільноти

**/team help** (сторінка) - там є уся інформація про створення та редагування спільнот

## Зовнішній вигляд

**/skin set (nickname)** - поставити скін по нікнейму. <br/>
**/skin url (url)** - поставити скін по посиланню. <br/>
*Обов’язково перезайдіть на сервер, щоб скін поновився*
